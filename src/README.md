Employee Management System (Console-Based)

A console-based Employee Management System developed using Core Java and JDBC that performs CRUD operations on employee records stored in a MySQL database.


Features

Add Employee
Update Employee
Delete Employee
Search Employee by ID
View All Employees



Technologies Used

Java (Core Java)
JDBC
MySQL
OOP Concepts
DAO Design Pattern



Project Structure

com.ems
├── dao
├── model
├── util
└── main



Database

Database: MySQL
Table: employee

Columns:
emp_id (Primary Key)
name
department
salary
email




How to Run

Clone the repository
Import project into IDE (VS Code / IntelliJ / Eclipse)
Configure MySQL database in DBConnection.java
Run MainApp.java





Key Concepts Applied

JDBC Connectivity
PreparedStatement
Exception Handling
Input Validation
Modular Design using DAO





Purpose
This project was built to strengthen Core Java, database connectivity, and object-oriented programming concepts for software development roles.
